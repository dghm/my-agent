報價產線

兩段式 + 印刷

啟動時機：新案需求討論完成，要對客戶提出服務報價。

  1. Skill：quote-step1-md — 整理需求成報價草稿 MD
    Claude 會把討論內容整理成標準化報價草稿。
    新客戶要報價，幫我整理報價草稿
  2. 人工確認 MD
    核對金額、範圍界定、付款條件、報價有效期限。
  3. Skill：quote-step2-html — 套 DGHM CI 出正式報價單
    鎖定版樣式：Header Banner、深藍頁尾條、淺橘提示框，不重新設計。
    MD 確認了，出報價單 HTML
  4. Skill：tools-pagination — A4 分頁 → 瀏覽器另存 PDF 套分頁，要印 PDF

合約產線

兩段式 + 印刷

啟動條件：報價單已確認，且 金額超過 NT$30,000 或工期超過 10 個工作天。

  1. Skill：contract-step1-md — 依報價單生成合約草稿 MD 依報價單生成合約草稿
  2. 人工確認草稿
    核對付款期程、交付項目、驗收條件、違約與智財條款。
  3. Skill：contract-step2-html — 出 DGHM CI 正式合約書 合約 MD 確認了，出 HTML
  4. Skill：tools-pagination — A4 分頁 → PDF（一式兩份用印） 套分頁，要印 PDF

User Guide 產線

兩段式 · 套客戶品牌

啟動時機：要為客戶的系統／模組產出操作手冊或 SOP 文件（如 hs128）。注意：這條線套的是客戶品牌 tokens.css，不是 DGHM CI。

  1. Skill：user-guide-step1-md — 整理內容大綱 MD
    章節結構、步驟教學、欄位說明、截圖位置。
    幫我做使用手冊，先出大綱
  2. 人工確認大綱
    核對章節順序、功能覆蓋是否完整、截圖是否備齊。
  3. Skill：user-guide-step2-html — 出 A4 正式操作手冊
    套客戶 --brand-* 變數與固定版型元件庫。
    大綱確認了，出 User Guide HTML
  4. 要印刷再走 tools-pagination（DGHM 對外文件時）
