DGHM Tools Dashboard

萬能數維有限公司 — 內部工具箱。點選卡片開啟對應的產生器（皆為單一 HTML 檔案，可直接於瀏覽器使用）。

品牌文件

客戶品牌風格指南產生器

填入客戶品牌色彩與視覺風格資訊，即時產生 tokens.css、guideline.html 與 Breakdance Colors JSON，供 DGHM 製作客戶文件時套色與排版參考。

 財務文件

應收帳款通知產生器

填寫請款資訊與項目明細，即時預覽並產生 A4 列印用的應收帳款通知 HTML，可直接複製、下載或開啟列印。

 財務文件

服務報價單產生器

填寫報價項目、方案勾選、總計與付款條件，即時預覽並產生多頁 A4 列印用的服務報價單 HTML，含簽署欄與有效期限提示。

 會議文件

既有客戶會議記錄

會議中先用快速記錄區隨手記下客戶提問與重點，會後一鍵整理進討論摘要，再產生可交付的會議紀錄 Markdown。

 業務文件

客戶需求 Brief 產生器

整理新客戶或新專案的需求訪談內容，產生結構化的 Brief 文件，方便團隊接續報價與規劃。

 客戶文件

操作手冊骨架產生器

製作客戶操作手冊（User Guide）前，填入基本資訊與章節規劃，即時產生結構化的 Markdown 文件骨架，之後交給 Claude Cowork 撰寫內容與 HTML 排版。

 財務文件

接案收入登記系統

登記每筆案件收入與收款狀態，依月／年彙整報稅用收入，並可一鍵帶入應收帳款通知產生器開請款單。

 內部管理

週工作排程規劃

填入本週客戶專案與內部工作的時間佔比，自動產生符合排程習慣的週表，輸出文字可貼到 Claude chat 建立 Google Calendar 事件。

 內部管理

Clear 風格待辦清單

模仿 iOS Clear App 的手勢操作：右滑完成、左滑刪除、拖曳排序、下拉新增。多個彩色清單，資料儲存於本機瀏覽器。

 行銷素材

微信 H5 名片產生器

填入個人與公司資訊，即時預覽微信風格的行動裝置電子名片，內含撥號、加微信、存名片動作。可下載成單一自足 HTML 檔（圖片內嵌）分享，或匯出 vCard 存入通訊錄。
